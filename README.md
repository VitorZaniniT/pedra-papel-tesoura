# pedra-papel-tesoura
Jogo de Pedra, Papel e Tesoura desenvolvido com HTML, CSS e JavaScript. O usuário escolhe uma das opções enquanto o computador gera uma escolha aleatória. O sistema compara as escolhas e exibe o resultado da partida.  Este projeto foi criado para praticar lógica de programação, geração de valores aleatórios e manipulação do DOM com JavaScript.
